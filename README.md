# Cactus Vault Prototype

Mike-style clickable prototype for the Cactus multifamily acquisition intelligence app.

## Product spine

Build The Vault → watch extraction populate live → explore table/map/chat → analyze a deal/address → select comps and enrichments → generate IC/bank outputs → later automate weekly hitlists.

## Prototype screens

1. Homepage
2. Create Vault
3. Live Extraction
4. Vault Table
5. Vault Map
6. Deal Analysis
7. Comps + Data
8. Outputs
9. Automations

## Local development

```bash
npm run dev
```

Open http://localhost:3000.

## Verification
